# Marble Contracts (Next.js + Tailwind)

Минималистичное тёмное веб-приложение для компании по укладке мрамора: главная страница с полем ввода **ID договора** и кнопкой **Поиск**.

## Запуск

Установите зависимости и запустите dev-сервер:

```bash
npm i
npm run dev
```

Откройте `http://localhost:3000`.
